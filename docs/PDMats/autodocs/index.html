    PDMats.invquad — Method.
    invquad(a::AbstractPDMat, x::StridedVecOrMat)

    Return the value of the quadratic form defined by inv(a) applied to x.

    For most PDMat types this is done in a way that does not require evaluation of inv(a).

    If x is a vector the quadratic form is x' * a * x. If x is a matrix the quadratic form is applied column-wise.

    Allocating and in-place versions of the whitening transform (or its inverse) defined by a applied to x

    If the covariance of x is a then the covariance of the result will be I. The name whiten indicates that this function transforms a correlated multivariate random variable to "white noise".

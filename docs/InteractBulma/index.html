    InteractBulma

    Package to create simple web apps in Julia based on InteractBase. To install the package and its dependencies simply run the following code in the REPL:

    Pkg.clone("https://github.com/piever/InteractBulma.jl")
Pkg.build("InteractBulma")

    To learn how to use this package, check out the InteractBase documentation.

    Simply type using InteractBulma instead of using InteractBase and the same commands of InteractBase will work and the GUI will be styled with Bulma css.

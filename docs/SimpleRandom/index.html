    SimpleRandom.probs — Method.

    probs(X::RV) returns an iterator of the probabilities associated with the values in X.

    SimpleRandom.random_choice — Method.

    random_choice(X) for a random variable X returns a value of X according to its probability distribution. That is, the probability a value v is returned is X[v].

    SimpleRandom.random_choice — Method.

    random_choice(weights) randomly chooses a value from 1 to n, where n is the number of elements in weights. The probability that k is chosen is proportional to weights[k]. The weights must be nonnegative and not all zero.

    random_choice(dict) choose a random key k from dict with weight proportional to dict[k]. Thus, dict must be of type Dict{S, T<:Real}.

    SimpleRandom.random_subset — Method.

    random_subset is used to create random subsets as follows:

    • random_subset(A): random subset of A with each element

    chosen with probability 1/2.

    • random_subset(A,k): random k-element subset of A.
    • random_subset(n): random subset of 1:n.
    • random_subset(n,k): random k-element subset of 1:n.
    SimpleRandom.random_unit_vector — Method.

    random_unit_vector(d) returns a random d-dimensional unit vector.

    SimpleRandom.report — Method.

    report(X) prints out a list of the values of X and their associated probabilities

    SimpleRandom.validate! — Method.

    validate!(X) ensures that the probabilies of the values in X sum to one. If not, they are rescaled.

    SimpleRandom.vals — Method.

    vals(X::RV) returns an iterator of the values this random variable can take. Use X[v] to get the associate probability of the value v.

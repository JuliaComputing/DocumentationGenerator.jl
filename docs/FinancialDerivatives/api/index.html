Derivatives

FinancialDerivatives.AmericanOption — Type.
AmericanOption(s, k, r, σ, t, call)

American option.

Arguments

  • s: underlying price
  • k: strike price
  • r: risk-free interest rate
  • σ: volatility
  • t: time to expiration
  • call: 1 if call, -1 if put

FinancialDerivatives.FXOption — Type.
FXOption(s, k, r_d, r_f, σ, t, call)

Foreign exchange option.

Arguments

  • s: underlying price
  • k: strike price
  • r_d: domestic risk-free interest rate
  • r_f: foreign risk-free interest rate
  • σ: volatility
  • t: time to expiration
  • call: 1 if call, -1 if put
FinancialDerivatives.InterestRateDerivative — Type.
InterestRateDerivative(k, r, σ, θ, t)

Interest rate derivative.

Arguments

  • k: speed of reversion
  • r: initial interest rate
  • σ: instantaneous volatility
  • θ: long term mean level
  • t: time interval

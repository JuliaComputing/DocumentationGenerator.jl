    LorentzVectors.LorentzVector — Type.
    LorentzVector(t, x, y, z)

    Lorentz 4-vector, as used in Special Relativity.

    The metric convention is g = diag(+1,-1,-1,-1). No distinction is made between co- and contravariant vectors.
